## 仿网易云web网站

### 简介

>该项目是模仿网易云web网站以练习前端和后端的练手项目

### 技术栈

>vue + vue-router + vuex + express + element-ui

### 已完成功能

* 音乐播放、音乐列表添加
* 查看首页热门歌单、播放歌单音乐
* 查看热门榜单曲、播放音乐

### 安装运行

    git clone https://github.com/842686697/cloud-music.git
    cd music-cloud
    npm run serve
